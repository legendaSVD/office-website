# ZIZIYI Office

![Логотип проекта](./public/logo.svg)

ZIZIYI Office — локально-ориентированное веб-приложение для просмотра и редактирования файлов Word, Excel и PowerPoint прямо в браузере. Проект делает акцент на приватности: документы обрабатываются на клиенте и сохраняются локально.

## ✨ Особенности
- Открывает и создаёт файлы `.docx`, `.xlsx` и `.pptx`.
- Работает по принципу local-first и использует локальное хранилище браузера.
- Поддерживает быстрый старт редактора через параметры URL.
- Использует современный стек Next.js 15 и клиентское состояние через Zustand.
- Подготовлен к сборке и деплою в Cloudflare Pages.

## 🚀 Быстрый старт
### Требования
- Node.js 22+
- `pnpm`

### Установка
```bash
pnpm install
```

### Запуск в разработке
```bash
pnpm dev
```

### Сборка
```bash
pnpm build
```

## 📁 Структура проекта
- `app` — маршруты и страницы приложения на базе App Router.
- `components` — переиспользуемые UI-компоненты и SVG-элементы.
- `lib` — вспомогательная логика для работы с редактором и данными.
- `messages` — локализационные сообщения интерфейса.
- `public` — публичные статические ресурсы, включая логотипы и SVG.
- `store` — глобальное состояние клиента.

## 🛠 Использование
### Локальный запуск
Откройте `http://localhost:3000` после старта dev-сервера.

### Быстрое создание документа
```text
http://localhost:3000/editor?new=docx
http://localhost:3000/editor?new=xlsx
http://localhost:3000/editor?new=pptx
```

### Деплой
Для сборки статического артефакта используется `pnpm build`, а для публикации в Cloudflare Pages в `package.json` уже предусмотрены команды `pnpm deploy` и `pnpm deploy:preview`.

## 🧪 Технологии
- Next.js 15
- React
- TypeScript
- Zustand
- IndexedDB
- Cloudflare Pages
